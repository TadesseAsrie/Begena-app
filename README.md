#  Begena Ethiopian Traditional Music Instrument Simulator

A modern, interactive web-based **Begena (Ethiopian Traditional Harp) Simulator** built using **HTML5, CSS3, and Vanilla JavaScript**. This project recreates the experience of playing the Begena through realistic string interactions, dynamic sound generation using the Web Audio API, and an elegant Ethiopian-inspired user interface.

---

##  Overview

The Begena is one of Ethiopia's oldest and most respected traditional musical instruments. Often called the **"Harp of King David,"** it is widely used for spiritual reflection, meditation, and traditional performances.

This simulator allows users to experience the sound and interaction of the Begena directly in their web browser without requiring any external software or installation.
